This project is to determine "weird" items using reviews data from http://jmcauley.ucsd.edu/data/amazon/

To run: spark-submit parse_json2.py <url with json.gz from this website> 

Ex: spark-submit parse_json2.py http://snap.stanford.edu/data/amazon/productGraph/categoryFiles/reviews_Toys_and_Games_5.json.gz
